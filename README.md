‎README – Password Strength Checker
‎
‎A simple and easy-to-use tool that'll help you in creating a password. It checks length, characters, variety, and how long it can take for a computer to crack the code. It can also give suggestions on what to add to your password to make it harder to crack.
‎
Features
‎
‎Checks length and character types.
‎ex. lowercase, uppercase, etc
‎
‎Spots easy to see patterns like repeated characters or keyboard sequences
‎
‎Estimates how long would it take for a computer to crack the code
‎
‎Gives short, clear tips to strengthn your password
‎
‎Simple strength score: Weak, Okay, Strong, Excellent
‎
‎Counts how many digits and symbols are used
‎
‎Checks for consecutive letters or numbers (like abcd or 1234)
‎
‎Counts the unique characters in the password for variety
‎
‎Detects if password contains anything unusual
‎
‎
‎How to Use
‎
‎1. Type a password.
‎
‎
‎2. The tool scans it and checks everything listed above.
‎
‎
‎3. You get a strength score and quick tips, sometimes including what patterns to avoid.
